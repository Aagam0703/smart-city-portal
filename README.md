# Smart City Information Portal

A centralized platform for citizens, administrators, and stakeholders to access city data and insights.

## Features
- Real-time data dashboards
- Public datasets integration
- RESTful API endpoints
- Automated data processing
- Secure admin authentication

## Technology Stack
- Python 3.8+
- Django Web Framework
- PostgreSQL Database
- Celery for task scheduling
- Plotly/Dash for visualizations
- Docker for deployment

## Quick Start

1. Clone the repository
2. Copy `.env.example` to `.env` and configure
3. Run `docker-compose up --build`
4. Access the application at `http://localhost:8000`

## Development

See `requirements.txt` for Python dependencies and `docker-compose.yml` for service configuration.
